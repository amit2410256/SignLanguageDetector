# 🤟 Sign Language Detector

## B.Tech Mini Project

### 📌 Project Description
This project aims to develop a real-time Sign Language Detection System using Artificial Intelligence and Computer Vision. The system will detect hand gestures through a webcam and convert them into text and speech.

## 🎯 Project Goal

Develop an AI-powered application that:
- Detects sign language in real time.
- Converts gestures into text.
- Converts text into speech.
- Stores prediction history.
- Provides a user-friendly web dashboard.
